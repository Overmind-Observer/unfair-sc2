# CRITICAL STRIKE

*Added in 1.32*

Every unit has a change to do a critical hit:

| Chance | Damage multiplier |
|-------------|-----------------|
| 20% | х2 |
| 30% | х2 |
| 40% | х2 |
| 20% | х3 | 
| 30% | х3 |
| 40% | х3 |
| 20% | х4 |
| 30% | х4 |
| 40% | х4 |

Set to each player individually.
