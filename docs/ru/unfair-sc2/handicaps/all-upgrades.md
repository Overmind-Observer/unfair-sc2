# ALL UPGRADES

Изначально дает игроку все возможные улучшения, доступные в многопользовательском режиме СК2. 

Не работает на улучшения из гандикапа [EXTRA UPGRADES](./extra-upgrades.md)!

Дается каждому игроку индивидуально.
